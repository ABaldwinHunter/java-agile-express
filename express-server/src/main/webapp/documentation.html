Documentation

Install

Download the latest war file and deply it to your favouriute servlet container. The latest build is tested on Tomcat 6.0.20.

The latest build is configured by default for Postgres..

Configure

Most configuration is done in the application.properties file which is located in the exploded war file under WEB-INF/classes.

Use

Register

The fist thing you will need to do is register an user account.

Create a project

You will now need to create a project. When you create a project you automatically become it's manager.
