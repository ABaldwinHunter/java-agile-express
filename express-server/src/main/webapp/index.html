Express - Agile Project Management

Express brings a full suite of Agile project management tools to your project. Including backlog management, iteration management, virtual wall, and coming soon, defect management and risk/issue management. All of this comes to you in a single, seemless application without the need for complex and costly integration exercises.

Charting components assist in rapidly assessing iteration, project and developer progress
Wall view allows developers and project managers to quicly assess and manage iteration tasks
Coming soon full defect lifecycle management. Bring defects seemlessly into an iteraton for development like any other iteration task. Track progress and status, manage communication.
